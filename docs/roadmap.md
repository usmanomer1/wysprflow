# Roadmap

## Shipping now

- macOS Tauri desktop app
- first-run setup
- Deepgram speech-to-text
- OpenRouter and Anthropic cleanup
- paste injection with clipboard restore
- dictionary, snippets, run history
- settings for microphone, sounds, translation target, startup, and prompts

## Next

- direct text insertion path that does not rely on paste
- lower-latency cleanup
- stronger diagnostics and exportable debug reports
- better first-run success feedback after the first working dictation

## Later

- local-only transcription and cleanup mode
- broader provider support beyond key storage
- updater flow for packaged releases
- Windows support
